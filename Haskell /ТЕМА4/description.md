Название: transpose  
Тип: [[a]] -> [[a]] 
Описание: принимает 2d  матрицу и транспонирует строки и столбцы своего аргумента 
Примеры:  
1)   
transpose [[1,2,3],[4,5,6],[7,8,9]]  
Результат: [[1,4,7],[2,5,8],[3,6,9]]  
2)  
transpose ["hey","there","guys"]
Результат: ["htg","ehu","yey","rs","e"]  

Реализация:   
```
myTranspose :: [[a]] -> [[a]] 

```
